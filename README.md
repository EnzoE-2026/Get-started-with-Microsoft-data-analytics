# Notes

## Intro >> Unit 1/6

** Why do we need Data Analysts? **
- Businesses collect vast amounts of data (e.g., purchases, browsing behavior, aisle visits, spending habits)
- Despite this, many struggle to use data meaningfully
- The core issue: understanding the data and using it to drive action
- Job: To assist businesses in finding insights and uncovering hidden value in troves of data through storytelling

Data Analysts work closely with other data experts such as (data) scientists and engineers to get all necessary information

Data alone isn’t enough — the value comes from the story it tells and how a business can leverage it. Stories help decision-makers take faster, smarter actions. Reports should communicate clearly and support actionable insights.

### Data utilization (How the data gets used): 
- Track inventory and optimize pricing
- Detect customer trends and purchase patterns
- Recommend products and prevent fraud
- Analyze daily, weekly, monthly sales patterns
- Compare sales year-over-year


## Overview of data analysis >> Unit 2/6
Before data can be used to tell a story, it must be run through a process that makes it usable in the story. Data analysis is the process of: 

1. Identifying 
2. Cleaning
3. Transforming 
4. Modeling data

To discover meaningful and useful information. The data is then crafted into a story through reports to support the critical decision-making process.

To analyze data, core components of analytics are divided into the following categories:

- Descriptive
- Diagnostic
- Predictive
- Prescriptive
- Artificial Intelligence (AI)

### 1. Descriptive analytics - *What happened*?
- Summarizes large semantic models based on historical data to the stakeholders
- KPI’s (key performance indicators) are used to track/measure the success of or failure of objectives
- Metrics (very similar to KPI’s) such as return on investment (ROI) is a common metric used in many industries

Example : Generating report to view sales and financial data
